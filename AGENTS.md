# Agent Instructions for alerthq

alerthq is a monorepo for discovering and normalizing alert definitions across cloud providers (AWS CloudWatch, Datadog, Grafana, etc.).

## Standards and Conventions

All coding standards, project structure, canonical data rules, and post-edit verification checklists are in **[CONTRIBUTING.md](CONTRIBUTING.md)**. Read it before making changes.

## Workflows

- **Adding a new provider**: Read and follow `docs/workflows/add-provider/workflow.md` and its companion files (`templates.md`, `research.md`, `readme-template.md`).
- **Reviewing code against standards**: Read and follow `docs/workflows/review-standards/workflow.md`.
